MongoDBCommandPrompt.exe and MongoDBPortable.exe are generated with PortableApps.com Launcher

HOW-TO generate MongoDBPortable.exe 
- Run PortableApps.com Launcher
- Select MongoDBPortable\Source folder

HOW-TO generate MongoDBCommandPrompt.exe
- open MongoDBPortable\Source\App\AppInfo\appinfo.ini
- replace line AppID=MongoDBPortable with AppID=MongoDBCommandPrompt
- Run PortableApps.com Launcher
- Select MongoDBPortable\Source folder
- put back original AppID=MongoDBPortable in appinfo.ini file
